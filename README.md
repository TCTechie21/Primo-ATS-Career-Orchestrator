## 📖 Project Overview
*Primo ATS* is a "Universal Logic" engine designed to ingest any resume and any job description to bridge the gap between a candidate's current state and their target Cloud/IT role. 

Built on **AWS PartyRock**, this application uses advanced prompt orchestration to deliver a 360-degree career roadmap including technical gap analysis, automated STAR interview stories, and strategic business insight.

## 🛠️ Key Technical Features
* **Universal Data Ingestion:** Engineered dynamic extraction logic that identifies company identity and industry challenges from any raw, unformatted Job Description.
* **Architecture:** Refactored to a **Single Source of Truth** data flow where the application treats the uploaded CV as the primary variable to minimize AI hallucinations.
* **The "Uno Reverse" Strategy:** A unique feature that identifies the business pain points within a JD, allowing candidates to ask high-level strategic questions during interviews.
* **Technical Gap Analysis:** Automatically generates a 4-week learning roadmap based on the delta between the candidate's skills and the JD requirements.
